## Описание проекта.  
Требуется подготовить прототип модели машинного обучения. Модель должна предсказать коэффициент восстановления золота из золотосодержащей руды. В распоряжении имеются данные с параметрами добычи и очистки. Модель поможет оптимизировать производство, чтобы не запускать предприятие с убыточными характеристиками.  

## Используемые инструменты и библиотеки:  
*pandas*  
*sklearn*   
*numpy*   
*matplotlib*     
*seaborn*     

## Результаты проекта.  
При выполнении проекта исследована и отображена на графиках суммарная концентрация золота на разных стадиях технологического процесса добычи из руды. Далее созданы и протестированы раличные модели машинного обучения. Для оценки качества модели применялась метрика sMAPE-симметричное среднее абсолютное процентное отклонение, т.е. насколько предсказанные значения в среднем отличаются от истиных в процентном отношении. Лучшей оказалась модель RandomForestRegressor. На тестовой выборке эта модель показала результат sMAPE = 9.22.
