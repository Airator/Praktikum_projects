## Задачи проекта.
 Требуется построить модель, которая по фотографии определит приблизительный возраст человека. В распоряжении имеется набор фотографий людей с указанием возраста. 

## Используемые инструменты и библиотеки:  
*pandas*  
*keras*  
*numpy*  
*matplotlib*   
*seaborn*  
  

## Результаты проекта.  
 Построена модель на основе нейросети ResNet50. После тестирования модели применяемая метрика МАЕ показала результат 6.99.
